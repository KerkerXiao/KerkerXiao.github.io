注意：如果此时没有成功则进入 GitHub 仓库设置：

  • 打开你的 GitHub 仓库页面：https://github.com/KerkerXiao/KerkerXiao.github.io
  • 点击仓库页面右上角的 Settings。

配置 GitHub Pages：

  • 在设置页面的左侧栏中，找到并点击 Pages。
  • 在 Source 部分，确保选择了 master 或 main 分支，通常是 master，并且文件夹设置为 / (root)。
  • 确保没有选择 gh-pages 分支，因为默认情况下，用户和组织页面应使用 master 或 main 分支。
